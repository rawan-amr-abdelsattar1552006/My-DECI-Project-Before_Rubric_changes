Category #1:
	Name: Strong

	Description:
		length: > or = 12 charcters
		Includes all of the following types of charcters :
			1- numbers 
			2- upper case letters
			3- lower case letters
			4- special characters or symbols  

	Meet minimum security requirments? : Yes

Category #2:
	Name: Fair

	Description:
		length: > 8 charcters(8 and 12 not included int the range)
		Includes at least two of the following types of charcters :
			1- numbers 
			2- upper case letters
			3- lower case letters
			4- special characters or symbols 
			
		

	Meet minimum security requirments? : Yes
Category #3:
	Name: Weak

	Description:
		length: <= 8 charcters
			or at any length and :
		Includes only one of the following types of charcters:
			1- numbers 
			2- upper case letters
			3- lower case letters
			4- special characters or symbols 
		

	Meet minimum security requirments? : No,insecure password that needs to be re-created
